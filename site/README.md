# CertiCovid

## Requirements

- Linux/MacOS
- NodeJS v14.x or above
- Yarn ( `npm install --global yarn` )

## Getting Started

Install Dependencies
```sh
$ yarn install
```

Then, just run the development server
```sh
$ yarn dev
```

Finaly open [http://localhost:3666](http://localhost:3666) with your browser to see the result.
