## Docker Tools

### Requirements

- Linux/MacOS env
- Docker
- Docker Componse

### Move to correct Folder

```sh
$ cd tools/docker
```

### Configure Env Variables

Update values of `.env` file

### Build Docker Image

```sh
$ docker-compose build
```

### Start CertiCovid Containers

```sh
docker-compose up -d 
```

### Stop CertiCovid Containers

```sh
docker-compose down 
```
